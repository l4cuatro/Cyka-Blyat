Mobile goal lift is front of robot

--------------------------------------------------------

Motor Layout

1- lChain (left chain bar motor)
2- lyDrive (left y-cabled drive motors)
3- rLift (right DR4B motor)
4- lGoal (left mobile goal lift)
5- mlDrive (left middle drive motor)
6- lLift (left DR4B motor)
7- rGoal(right mobile goal lift)
8- mrDrive (right middle drive motor)
9- ryDrive (right y-cabled drive motors)
10- rChain (right chain bar motor)

--------------------------------------------------------

Sensor Layout

----------------------------

Analog

1- goalPot (mobile goal potentiometer)
2- liftPot (DR4B potentiometer)

----------------------------

Digital

1, 2- lDriveEnc (left drive encoder)
3, 4- rDriveEnc (right drive encoder)
5, 6- chainEnc (chain bar encoder)
